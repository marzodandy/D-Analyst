Data Cleaning

    Bike Sales Report Data Cleaning with Google Spreadhseets

    Utilized Google Spreadsheets to clean the Bike Sales Report.
    The usage of Google Spreadsheets aimed to be more efficient with fewer risks for minor data cleaning.

    Here's what I did:
    • Removed duplicate rows;
    • Filled out missing values by adjusting with the same value provided in the other column; and/or
    • Performed calculations between profit, cost, and revenue column to fill the blank values.

    Melbourne Property Sales Data Cleaning

    Cleaned a Melbourne Property Sales data report from 2016-2017.
    Utilized MySQL functions like START TRANSACTION, UPDATE, ALTER, etc.

    Here's what I did:
    • Re-sorted number column by setting it to NULL, then modifying the column with AUTO_INCREMENT and PRIMARY KEY function;
    • Reformatted the "Property Type" column for better data readability and analysis efficiency.
